## Työn aihe

Ohjelma tulee olemaan samankaltainen kuin ulkoministeriön matkustusilmoitustietokanta.
Matkustaja täyttää matkustusilmoituksen ja käyttäjä sekä ministeriön kriisipäivystäjä voivat selailla ilmoituksia.
